# Azure-Honeypot

## Objective

The Azure Honeypot project aimed to establish a controlled environment for detecting cyber attacks. The main focus was to ingest and analyze logs within the Microsoft Sentinel SIEM, generating live telemetry from real brute force attacks. This project was designed to deepen understanding of network security and attack patterns.

### Skills Learned
- Understanding of SIEM concepts and practical application with Sentinel
- Proficiency in analyzing Windows logs
- Ability to recognize attack signatures and patterns
- Experience writing KQL scripts to filter attacker information from log data
- Development of critical thinking skills in cybersecurity

### Tools Used
- Microsoft Azure for setting up the project and deploying a vunerable virtual machine
- Microsoft Sentinel for log ingestion and analysis
- IP database to map geolocation data
